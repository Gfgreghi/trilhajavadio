# Contador

Projeto desenvolvido em Java como exercício para praticar métodos, estruturas de repetição e tratamento de exceções.

## Como funciona

O programa solicita dois números inteiros.

- Se o segundo número for maior que o primeiro, é realizada uma contagem de acordo com a diferença entre eles.
- Se o primeiro número for maior que o segundo, é lançada uma exceção e o programa informa que o segundo parâmetro deve ser maior.

## Exemplo

**Entrada:**

```
5
8
```

**Saída:**

```
imprimindo o numero 1
imprimindo o numero 2
imprimindo o numero 3
```

## Conceitos praticados

- Scanner
- Métodos
- Laço `for`
- `try` e `catch`
- Exceções personalizadas (`throw` e `throws`)

## Objetivo

Praticar validação de parâmetros e o uso de exceções personalizadas em Java.
